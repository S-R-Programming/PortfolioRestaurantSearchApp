# レストラン検索アプリ(ポートフォリオ・デザインのみ)
Flutterでデザインが用意されていれば、そのUI通りに実装出来ることを示すためのポートフォリオ用のアプリです。なので実際に検索は実装していません。


## 開発環境
Android Studio Bumblebee | 2021.1.1 Patch 2<br>
macOS 12.2.1<br>
Flutter 2.10.4<br>
